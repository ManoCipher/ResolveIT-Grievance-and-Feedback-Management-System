# ResolveIT – AI-Assisted Grievance Management System

## Project Domain
EduTech 2.0 – Educational Administration & Student Support

## Description
ResolveIT is a role-based grievance management system designed for educational institutions.  
It integrates AI-assisted categorization, priority prediction, and analytics to improve transparency and resolution efficiency.

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Spring Boot
- Database: MySQL
- AI Module: NLP-based analysis
